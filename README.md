# Pokemon Tournament

**Project, Web-based Data Visualisation, Malmö University, Spring Semester 2026**

A dashboard for visualizing a fictional Pokémon league, built with vanilla JavaScript and D3.js.

## Work distribution

The project work was divided between the group members as follows:

- Sebastian worked on the home page and the gym spider chart.
- Sigge worked on the gym page (excluding gym spider chart).
- Markus worked on the Pokémon page.
